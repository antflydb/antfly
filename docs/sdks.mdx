# SDKs

Antfly provides type-safe client libraries for Go, TypeScript, Python, and Rust. All SDKs are auto-generated from the [OpenAPI specification](/docs/api) and kept in sync with every release.

| Language | Package | Registry | Source |
|----------|---------|----------|--------|
| Go | `github.com/antflydb/antfly-go/antfly` | [pkg.go.dev](https://pkg.go.dev/github.com/antflydb/antfly-go/antfly) | [GitHub](https://github.com/antflydb/antfly-go) |
| TypeScript | `@antfly/sdk` | [npm](https://www.npmjs.com/package/@antfly/sdk) | [GitHub](https://github.com/antflydb/antfly-ts) |
| Python | `antfly-sdk` | [PyPI](https://pypi.org/project/antfly-sdk/) | [GitHub](https://github.com/antflydb/antfly-py) |
| Rust | `antfly-client` | [crates.io](https://crates.io/crates/antfly-client) | [GitHub](https://github.com/antflydb/antfly/tree/main/rs/antfly-client) |

## Authentication

All three authentication methods are supported across SDKs: basic auth, API key, and bearer token.

## Quick Start

### Insert Documents

<CodeTabs persistKey="sdk-lang" samples={[
  {
    language: "Go",

  },
  {
